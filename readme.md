# award.css

`award.css` is a film awards logotype.

## Example

[http://var.tw/mfmftwtw/award.css/](http://var.tw/mfmftwtw/award.css/)

## Quick Start

1\. Include the file.

```html
  <link rel="stylesheet" href="award.css">
```

## Documentation

```html
<i class="award"></i>
```

|                   | Class Name         |
| ----------------- | ------------------ | 
| award effect      | `award`            |
| large             | `award-lg`         |
| small             | `award-sm`         |
